A painless workflow for managing large-scale machine learning experiments

Haven is a library for building, managing and visualizing large-scale reproducible experiments. It helps developers establish a workflow that allows them to quickly prototype a reliable codebase. It also helps them easily scale that codebase to one that can run, manage, and visualize thousands of experiments seamlessly.

Some Unique Features

Standardised Code Structure

Easy to get start and simple to use by building a codebase structure that can statisfy almost any machine learing projects

Visualize Experiment Results

Provide users more power to do whatever they want with the data through custom functionalities, producing desired results

Increased Launching Speed

Run experiments faster in parallel using job schedulars in the Cluster no matter what your computer configuration is

Better Experiment Management

More comfortable user experience by having functions such as filtering, resuming and restarting experiments. No worry about suddent accident!

Well-implemented Benchmarks

Reduce your workload. Grab existing benchmarks on GitHub for your project with minimum effort to reproduce or integrate current work

Version Control

Train your experiments as usual but you will never lose it!
